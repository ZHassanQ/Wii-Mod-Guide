    Canale Open Shop (Hombrew Browser)

    Per supporto in Inglese, chiedi su RiiConnect24 su Discord.

    Se hai bisogno di aiuto per qualsiasi cosa riguardo questo tutorial, per favore unisciti al server Discord di Open Shop Channel (consigliato) o inviaci un’email a [email protected].

    Homebrew Browser, che è quello su cui si basa il Canale Open Shop, funziona ma può essere buggato. Puoi anche ottenere app homebrew usando osc-dl e dal sito di Open Shop Channel.

    Il Canale Open Shop è dove puoi andare per ottenere app homebrew. Questo è un progetto di rinascita per un’applicazione chiamata Homebrew Browser.

    Di cosa hai bisogno

    • Una scheda SD o dispositivo USB
    • Homebrew Browser

    Istruzioni

    1. Estrai l’Homebrew Browser e mettilo nella cartella apps sulla tua scheda SD o dispositivo USB.
    2. Inserisci la tua scheda SD o dispositivo USB dentro la tua Wii. Ora puoi caricare l’Homebrew Browser dall’Homebrew Channel se vuoi.

    Applicazioni consigliate

    Ecco delle applicazioni consigliate che puoi ottenere sul Canale Open Shop:

    • CleanRip - Questo è un tool per dumpare i giochi Wii e GameCube. Visitala nostra pagina per più dettagli.
    • GCMM - Questo è un manager di memory card per GameCube per la tua console Wii.
    • MyMenuify Mod - Questo è un tool per installare temi per il tuo Menu Wii. Visita la nostra pagina per istruzioni su come usarlo.
    • Nintendont - Questo è un game loader per la tua Wii. Per maggiori informazioni, vedi questo thread GBAtemp.
    • SaveGame Manager GX - Questo è un manager dei salvataggi per la tua Wii. Ti permette di copiare i file di salvataggio e Mii alla e dalla tua Wii.
    • USB Loader GX - Questo è un USB Loader per la tua Wii. Visita la nostra pagina per istruzioni su come usarlo.
    • WiiFlow Lite - Questo è un altro USB Loader per la tua Wii. Visita la nostra pagina per istruzioni su come usarlo.
    • WiiMC-SS - Questo è un lettore multimediale per la tua Wii. Supporta film, musica, foto, stazioni radio, YouTube, e altro ancora.
    • WiiXplorer-SS - Questo è un manager di file per la tua Wii. Ti permette di accedere ai tuoi file sulla scheda SD, sul dispositivo USB e altro ancora.
    • YABDM - Questo è un tool per dumpare i contenuti installati nella tua Wii su file WAD. Visita la nostra pagina per istruzioni su come usarlo.
