    Wii Mod Lite

    Pour de l'assistance en anglais, veuillez demander de l'aide sur le serveur Discord de RiiConnect24.

    Table des matières

    • Ce dont vous avez besoin
    • Instructions
      • Section I - Téléchargement
      • Section II - Exécution

    Si vous avez besoin d’aide pour quoi que ce soit concernant ce didacticiel, veuillez rejoindre le serveur RiiConnect24 Discord (recommandé) ou [ envoyez-nous un e-mail à [email protected] ](mailto: support @ riiconnect24 .net).

    Ce tutoriel vous expliquera comment installer des WADs sur votre Wii avec Wii Mod Lite. Nous recommandons le gestionnaire de WADs Wii Mod Lite, mais il en existe d’autres tels que YAWMM.

    Nous déconseillons d’utiliser “WAD Manager” de Waninkoko.

    Ce dont vous avez besoin

    • Une carte SD ou un périphérique USB
    • Wii Mod Lite

    Instructions

    Section I - Téléchargement
    1. Décompressez Wii Mod Lite et placez-le dans le dossier apps sur votre carte SD ou votre périphérique USB.
    2. Insérez votre carte SD ou votre périphérique USB dans votre Wii, et lancez Wii Mod Lite depuis Homebrew Channel.
    Section II - Exécution

    Servez-vous de la croix directionnelle pour utiliser cet outil.

    1. Sélectionnez WAD Manager.
    2. Choisissez Emplacement SD Wii ou Périphérique de stockage de masse USB, selon que vous souhaitez utiliser une carte SD ou une clé USB.
    3. Sélectionnez le WAD que vous voulez installer. Dans cet exemple, nous installons l’IOS patché nécessaire pour RiiConnect24. Vous pouvez sélectionner plusieurs WADs en appuyant sur le bouton +.
    4. Appuyez deux fois sur “A” pour commencer l’installation.
    5. Si le message Finishing installation... OK! s’affiche, le WAD s’est installé correctement.
