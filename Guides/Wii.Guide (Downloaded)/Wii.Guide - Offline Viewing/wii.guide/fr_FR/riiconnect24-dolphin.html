    RiiConnect24 pour l’émulateur Dolphin

    Pour de l'assistance en anglais, veuillez demander de l'aide sur le serveur Discord de RiiConnect24.

    Table des matières

    Ce tutoriel va vous aider à installer RiiConnect24 sur l’émulateur Dolphin.

    Si vous avez besoin d’aide concernant ce tutoriel, veuillez contacter directement KcrPL#4625 sur Discord, rejoignez le serveur Discord de RiiConnect24 (recommandé, en anglais) ou envoyez-nous un e-mail à [email protected].

    Ce guide est seulement pour l’émulateur Dolphin..

    • Suivez ce tutoriel si vous voulez installer RiiConnect24 sur ton Wii.
    • Suivez ce tutoriel si vous voulez installer Riiconnect24 sur vWii (Mode Wii sur la Wii u).

    N’INSTALLEZ PAS RIICONNECT24 SUR UNE WII MINI ! Cela ne fonctionnera pas et cela briquera le système.

    Ce dont vous avez besoin

    • Un ordinateur avec Windows 10 ou une version plus récente ou tout système basé sur Unix
    • Dolphin
    • RiiConnect24 Patcher

    Instructions

    Section I - InstallerDolphin

    Si vous avez déjà installé l’émulateur Dolphin, passer au Section II

    1. Téléchargez la dernière version bêta de Dolphin, mais pas une version stable car elle est très obsolète !
    2. Extraire le fichier .7z en utilisant un programme comme 7-Zip ou WinRAR.
    3. Lancez Dolphin.
    4. Dans Dolphin, allez dans Outils -> Effectuez une mise à jour du système en ligne -> Choisissez votre région.

    Si vous avez une console Wii homebrewée , vous pouvez utiliser un dump BootMii NAND au lieu d’installer les fichiers système avec l’option Mise à jour système. Voir cette page pour plus d’informations.

    Section II - Installation de RiiConnect24

    Si vous utilisez une version de Windows avant Windows 10, vous ne pouvez malheureusement pas utiliser la WiiConnect24 dans Dolphin. Vous pouvez utiliser ce programme pour l’exécuter à la place.

    1. Cliquez sur le lien ci-dessus pour accéder à la page GitHub où se trouve le patcher.
    2. Téléchargez RiiConnect24Patcher.bat si vous êtes sous Windows, et RiiConnect24Patcher.sh si vous êtes sur un système Unix
    3. Sur Windows ouvrez Riiconnect24Patcher.bat. Sur les systèmes Unix, ouvrez le terminal et tapez bash, puis faites glisser RiiConnect24Patcher.sh dans le terminal et appuyez sur Entrée. Cela devrait ressembler à ceci bash RiiConnect24Patcher.sh.
    4. Appuyez sur 1 pour choisir “Start” et confirmez votre sélection en appuyant sur ENTER. (NOTE : Ces captures d’écran proviennent de la version Windows du patcher.)
    5. Pour ce guide, choisissez “Installer RiiConnect24 sur votre émulateur Dolphin”
    6. Choisissez “Express (Recommendé)”. Cela vous procurera tout ce dont vous aurez besoin.
    7. Sélectionnez votre région.
    8. Pendant que vous y êtes, RiiConnect24 Patcher peut également télécharger d’autres chaînes optionnelles qui n’utilisent pas RiiConnect24. [X] Représente l’option sélectionnée. Appuyez simplement sur 5 et ENTRÉE si vous n’êtes pas intéressé. ![Salons optionnels supplémentaires](/images/RC24_Patcher/6.JPG
    9. Appuyez sur 1 puis ENTRER pour commencer à patcher.
    10. Soyez patient…
    11. Après que ça se termine, on apprécié so vous prenez une minute pour nous envoyer un commentaire anonyme. Si vous ne voulez pas, fermez le patcher. Tous les fichiers dois être sur votre carte SD.
    12. Dans Dolphin, allez dans Outils puis Installez WAD. Installez tous les fichiers WAD un par un.
    13. Décompressez AnyGlobe Changer v1.0 et exécutez le fichier boot.dol. Sélectionnez votre région et sauvegardez. Cela permettra à la Chaîne Vote de fonctionner.

    Malheureusement, la Wii Mail ne fonctionne pas encore dans Dolphin.

    Vous devrez lancer la Chaîne Vote 4 fois au plus pour qu’elle commence à fonctionner.

    Si vous obtenez d’autres erreurs avec la Chaîne de Prévisions ou la Chaîne Infos, comme un code d’erreur commençant par FORE ou NEWS ou un message interrompu, vous pouvez essayer de supprimer vos VFFs avec RC24-Clear-Tool.
