    Wii Backup Manager

    Wii Backup Manager是一个将Wii游戏从电脑复制到Wii的软件，以便从Wii上玩游戏。 它也可以从GameTDB下载封面并列出您拥有的游戏。 此教程将告诉您如何将游戏从电脑复制到U盘。 其它功能也易于使用。

    你需要的是

    • 一个U盘
    • Wii Backup Manager

    如果您使用Mac，我们推荐使用Witgui，或适用于Windows、MacOS、Linux的Wii Backup Fusion。 Wii Backup Manager只兼容Windows。

    我们推荐用USB硬盘连接Wii玩游戏。 我们将在本指南中称它为”U盘”。 只要它们有USB2向后兼容，在市场上找到的大部分硬盘都可以工作。 容量1TB及以上的硬盘会正常工作。 U盘也可以工作，但由于大小它将无法存储很多游戏。

    确保U盘格式化为FAT32或NTFS。 不要格式化为exFAT、extFS或WBFS，后者是存储Wii游戏的旧格式。

    步骤

    第一章 - 下载
    1. 解压Wii Backup Manager并运行。
    2. 将U盘插入电脑。
    第二节 - 复制游戏
    1. 转到磁盘1 标签，选择磁盘。
    2. 选择为Wii游戏使用的硬盘。
    3. 转到文件标签，选择添加。
    4. 选择多个文件以添加多个游戏或选择文件夹添加游戏文件夹。
    5. 选择传输，然后选择磁盘1来传输游戏。 传输游戏可能需要一些时间。
    一旦完成设置

    继续cIOS
    cIOS是使用USB Loader的必需品。

    继续USB Loader GXUSB Loader GX是一个USB Loader，可以让Wii从U盘玩游戏。

    继续WiiFlow WiiFlow是一个USB Loader，可以让Wii从U盘或SD卡玩游戏。
