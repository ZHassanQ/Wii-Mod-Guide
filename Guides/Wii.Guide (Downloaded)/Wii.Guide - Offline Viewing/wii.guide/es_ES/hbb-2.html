    Canal Open Shop (Homebrew Browser)

    Si tienes problemas para seguir los pasos de esta guía, por favor solicita ayuda en nuestro servidor de Discord (recomendado), o envíanos un correo electrónico. Soporte disponible solo en inglés.

    Si necesitas ayuda para algo relacionado con este tutorial, únete al servidor de Discord del Open Shop Channel (recomendado) o envíanos un correo electrónico a [email protected]t.

    Homebrew Browser, el software en el que el Canal Open Shop está basado, funciona pero puedes tener bugs. También puedes obtener aplicaciones homebrew con osc-dl o el sitio web del Canal Open Shop.

    El Canal Open Shop es donde puedes obtener aplicaciones homebrew. Es un proyecto para revivir una aplicación llamada Homebrew Browser.

    Requisitos

    • Una tarjeta SD o unidad USB
    • Homebrew Browser

    Instrucciones

    1. Extrae el Homebrew Browser a la carpeta apps en tu tarjeta SD o unidad USB.
    2. Conecta la tarjeta SD o unidad USB a tu Wii. Ahora puedes ejecutar el Homebrew Browser desde el Canal Homebrew.

    Aplicaciones recomendadas

    Aquí puedes encontrar algunas aplicaciones recomendadas que puedes obtener en el Canal Open Shop:

    • CleanRip - Es una herramienta para volcar juegos de Wii o GameCube. Visita nuestra página para más detalles.
    • GCMM - Es un gestor de tarjetas de memoria GameCube para tu Wii.
    • My Menuify Mod - Es una herramienta para instalar temas en tu Menú de Wii. Visita nuestra página para las instrucciones de uso.
    • Nintendont - Es un cargador de juegos de GameCube para tu Wii. Para más información, por favor visita este hilo de GBAtemp.
    • SaveGame Manager GX - Es un gestor de partidas guardadas para tu Wii. Esto te permite copiar partidas guardadas y Miis de/a tu Wii.
    • USB Loader Gx - Es un cargador USB para tu Wii. Visita nuestra página para las instrucciones de uso.
    • WiiFlow Lite - Es otro cargador USB para tu Wii. Visita nuestra página para las instrucciones de uso.
    • WiiMC-SS - Esto es un reproductor multimedia para tu Wii. Soporta películas, música, fotos, emisoras de radio, YouTube, y más.
    • WiiXplorer-SS - Este es un gestor de archivos para tu Wii. Te permite acceder a tus archivos en tu tarjeta SD, dispositivo USB y más.
    • YABDM - Esta es una herramienta para volcar contenido instalado en tu Wii a archivos WAD. Consulta nuestra página para obtener instrucciones sobre cómo usarla.
